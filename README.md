Privacy Policy for the iOS app “又是一天”

# Privacy Policy

“又是一天”非常重视您的隐私。

## Data Collection
本应用不会收集、存储、共享或上传任何个人数据。

所有用户记录的内容仅保存在设备本地，不会被发送至任何服务器，
开发者无法访问用户的任何数据。

## Third-Party Services
本应用不使用任何第三方分析工具、广告 SDK 或追踪服务。

## Network Usage
本应用不要求用户注册账号，不进行用户身份识别。
